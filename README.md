# Quantum System Simulator UI

Учебное Django-приложение для визуальной сборки простых квантовых систем и запуска численного моделирования через библиотеку QuTiP.

Проект ориентирован на образовательный сценарий: студент или преподаватель может задать уровни энергии, взаимодействие с излучением, начальное состояние системы и затем посмотреть эволюцию населённостей и других наблюдаемых величин во времени.

## Возможности

- визуальный редактор энергетических уровней и переходов;
- задание взаимодействия с излучением через стрелки между уровнями;
- независимые единицы измерения для энергий, отстроек, частот Раби и ширин линий;
- ввод начального состояния в виде QuTiP-кода;
- ввод пользовательских наблюдаемых операторов;
- ввод дополнительных операторов, описывающих диссипацию энергии и декогеренцию;
- расчёт эволюции с помощью `qutip.mesolve`;
- графики населённостей и наблюдаемых величин;
- простая циклическая анимация изменения населённостей;
- сохранение систем и запусков в `SQLite`;
- экспорт конфигурации системы и результатов расчёта в `JSON` и `CSV`.

## Основной пользовательский сценарий

1. Открыть страницу `Рабочая область`.
2. Создать уровни энергии и переходы между ними.
3. Настроить параметры взаимодействия с излучением.
4. Сохранить систему.
5. Задать начальное состояние через QuTiP-выражение.
6. При необходимости выбрать уровни для графиков и задать дополнительные наблюдаемые.
7. Запустить расчёт.
8. Посмотреть графики, анимацию и при необходимости экспортировать результат.

## Хранение данных

В проекте используются две основные модели:

- `QuantumSystem` — сохранённая конфигурация квантовой системы;
- `SimulationRun` — отдельный запуск расчёта с параметрами и результатами.

## Примеры выражений QuTiP

Вектор состояния:

```python
basis(2, 0)
```

Матрица плотности:

```python
fock_dm(5, 2)
```

Наблюдаемый оператор:

```python
proj_excited = basis(2, 1) * basis(2, 1).dag()
```

Дополнительный оператор, описывающий диссипацию:

```python
c_custom = np.sqrt(2 * np.pi * 0.05e6) * basis(3, 0) * basis(3, 2).dag()
```

## Учебная ценность проекта

Приложение помогает студентам:

- визуально связать энергетические уровни с численной моделью;
- понять роль частоты Раби, отстройки и ширины линии;
- перейти от физической схемы к гамильтониану в rotating wave approximation;
- увидеть, как QuTiP используется в прикладной задаче моделирования открытых квантовых систем.
